# WTD - What To Do?

## A Neat Application To Manage Your To-Do Lists.

* One folder can only have one to-do file.
* Running wth without any subcommands executes the print function.

Dependecies:

- [Python3](https://www.python.org/downloads/)

Usage:

```
wtd [SUBCOMMAND] <ARGS>
```
